# base-python-poetry

A Python (Debian) Docker image with poetry. The tag corresponds to the python version

List of installed packages:

* curl
* git
* build-essential
* zip unzip
* wget
* nano
* libffi-dev
* libfreetype6-dev
* libfribidi-dev
* libharfbuzz-dev
* libjpeg-turbo-progs
* libjpeg62-turbo-dev
* liblcms2-dev
* libopenjp2-7-dev
* libtiff5-dev
* libwebp-dev
* libssl-dev
* freetds-dev
* freetds-bin
* unixodbc-dev
* tdsodbc

Get this image on:
